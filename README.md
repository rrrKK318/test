# Lightweight AI Conversation Error Annotation Tool (v0.1)

Internal workspace prototype for importing AI conversation text and creating **Error Group** annotations.

## Run
Open `index.html` in a browser.

## Included deliverables
- Working prototype: `index.html`, `app.js`, `styles.css`
- Sample imported text: `sample_conversation.txt`
- Sample exports: `sample_export_report.md`, `sample_export.jsonl`, `sample_error_groups.csv`, `sample_spans.csv`
- v0.2 notes: `V0_2_NOTES.md`

## Scope
- Import screen (plain text/markdown/file upload)
- Annotation workspace with #n target and *n related context bound to same Error Group
- Error type manager with defaults E1-E7 + custom types
- Export screen generating Markdown, CSV tables, and JSONL
